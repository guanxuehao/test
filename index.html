一、向量与矩阵（线性代数的基础）

1.1 核心概念

  • 向量：n维有序数组，分为行向量、列向量，代表n维空间中的点/向量
  • 矩阵：m×n个数排成的m行n列的数表，记为Aₘₙ
  • 特殊矩阵：单位矩阵E、零矩阵O、对角矩阵、对称矩阵、正交矩阵
  • 矩阵的秩：矩阵中最高阶非零子式的阶数，记为r(A)

1.2 核心运算

  • 矩阵加法：同型矩阵对应元素相加
  • 数乘矩阵：数k乘以矩阵每个元素
  • 矩阵乘法：Aₘₙ × Bₙₚ = Cₘₚ（第一个矩阵列数=第二个矩阵行数）
  • 矩阵转置：Aᵀ（行变列、列变行），性质：(AB)ᵀ = BᵀAᵀ
  • 矩阵求逆：可逆矩阵A满足AA⁻¹=A⁻¹A=E，仅方阵可能可逆
可逆矩阵判定：|A| ≠ 0（行列式非零），r(A) = n（满秩）

二、线性方程组（核心应用场景）

2.1 方程组形式

一般形式：AₘₙX = b（非齐次）；AₘₙX = 0（齐次）

齐次方程组AX=0：总有解（至少零解）；非零解 ⇨ r(A) < n
非齐次方程组AX=b：有解 ⇨ r(A) = r(A|b)；唯一解 ⇨ r(A)=r(A|b)=n；无穷解 ⇨ r(A)=r(A|b)<n

2.2 求解方法

  • 消元法：通过初等行变换将增广矩阵(A|b)化为行阶梯形/行最简形
  • 基础解系：齐次方程组AX=0的解空间的一组基，含n-r(A)个线性无关的解向量
  • 通解结构：非齐次通解 = 齐次通解 + 非齐次特解

三、特征值与特征向量（矩阵的核心属性）

3.1 定义

对于n阶方阵A，若存在数λ和非零向量α，使得Aα = λα，则λ为特征值，α为对应特征向量

特征方程：|λE - A| = 0（特征多项式）；特征向量：(λE - A)X = 0的非零解

3.2 核心性质

  • 特征值之和 = 矩阵迹（主对角线元素和）：∑λᵢ = tr(A) = a₁₁+a₂₂+…+aₙₙ
  • 特征值之积 = 矩阵行列式：∏λᵢ = |A|
  • 相似矩阵：P⁻¹AP = B ⇒ A与B特征值相同
  • 矩阵对角化：A可对角化 ⇨ A有n个线性无关的特征向量
实对称矩阵：特征值全为实数，不同特征值的特征向量正交，必可对角化

四、线性变换（线性代数的几何本质）

4.1 核心概念

  • 线性变换：T(α+β)=T(α)+T(β)，T(kα)=kT(α)（保持线性运算）
  • 线性变换的矩阵表示：在一组基下，线性变换对应唯一矩阵
  • 基变换与坐标变换：不同基下，同一向量的坐标满足X = PX'（P为过渡矩阵）

4.2 几何意义

  • 矩阵乘法 = 线性变换的复合
  • 特征向量 = 线性变换下方向不变的向量（仅伸缩）
  • 正交变换：保持向量长度、夹角不变（如旋转、镜像）

五、二次型（二次齐次多项式的矩阵表示）

5.1 定义与表示

二次型：f(x₁,x₂,…,xₙ) = ∑∑aᵢⱼxᵢxⱼ（i,j=1到n），矩阵表示：f = XᵀAX（A为对称矩阵）

标准形：f = λ₁y₁² + λ₂y₂² + … + λₙyₙ²（仅含平方项）

5.2 核心考点

  • 化标准形方法：配方法、正交变换法（常用）
  • 惯性定理：标准形中正负惯性指数（正负特征值个数）唯一
  • 正定二次型：对任意非零X，XᵀAX > 0 ⇨ A的特征值全正 ⇨ 各阶顺序主子式全正

六、线性代数核心应用场景

  • 机器学习：特征降维（PCA）、线性回归、矩阵分解（SVD）
  • 图像处理：图像变换、压缩（基于矩阵运算）
  • 工程计算：电路分析、力学系统建模、信号处理
  • 数据分析：线性方程组求解、矩阵求逆用于参数估计

核心公式速查

矩阵求逆：A⁻¹ = (1/|A|)A*（A*为伴随矩阵）
行列式展开：|A| = ∑aᵢⱼAᵢⱼ（按第i行展开）
特征值和：∑λᵢ = a₁₁+a₂₂+…+aₙₙ
秩的性质：r(AB) ≤ min{r(A), r(B)}
正交矩阵：AᵀA = AAᵀ = E ⇒ |A|=±1
齐次解空间维数：dim = n - r(A)
